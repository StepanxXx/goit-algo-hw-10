# goit-algo-hw-10

## Обчислення визначеного інтеграла методом Монте-Карло

### Результати обчислень

| Метод                                 | Результат          | Похибка (абс.) |
| ------------------------------------- | ------------------ | -------------- |
| Функція `spi.quad` (аналітичний)      | 2.6666666666666665 | 2.96e-14       |
| Метод Монте-Карло (100 експериментів) | 2.664994400000001  | -              |

### Висновки

За результатами виконання обчислень бачимо, що метод Монте-Карло дає результат `2.66499`, який є дуже близьким до точного аналітичного значення інтеграла `2.66667` (обчисленого функцією `quad`).

Різниця між значеннями становить приблизно `0.00167`, або `0.06%`, що свідчить про високу точність методу Монте-Карло навіть при відносно невеликій кількості експериментів (у нашому випадку усереднення проводилося за 100 повторень по 100 000 точок).

Це підтверджує правильність реалізації алгоритму та ефективність методу Монте-Карло для наближеного обчислення визначених інтегралів, особливо у випадках, коли аналітичне вирішення є складним або неможливим.
